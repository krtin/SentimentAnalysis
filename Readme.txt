How to run?
1. Change working directory to code/
2. Run python main.py [--write] algo classes
3. Thats it!

--write -> will write results.csv in the base directory (optional)
--imbalance -> will use the imbalance data set, you can only use unigram, bigram in algo and 3 classes with this
algo -> Is used for selecting the algorithm, valid options are (unigram,bigram,tree,treemod and all)
classes -> Is used to choose the type of classification, valid options are (2 or 3)

*you can also use -h to see available options in command line
